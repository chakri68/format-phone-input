# Format Phone Number

## Vanilla JS

cd into the `./vanilla-js` directory first

```bash
cd ./vanilla-js
```

### Install dependencies

```bash
yarn
```

### Start the http-server

```bash
yarn dev
```

### To run jest tests

```bash
yarn test
```

## React (with vite)

cd into the `./format-tel-input` directory first

```bash
cd ./format-tel-input
```

### Install dependencies

```bash
yarn
```

### Start the vite server

```bash
yarn dev
```

### To run tests

```bash
yarn test
```
